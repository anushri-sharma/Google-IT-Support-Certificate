🔥 **Terminology: software = programs = applications**  
  – **Drivers**: software that enables interaction with hardware.  
  – **Applications**: tools for day-to-day job functions.  
  – **Utilities**: small tools like **Calculator**, **Settings**, and similar system utilities.

🔄 **Software is constantly changing**  
  – Developers release updates; software companies add or remove features and change behavior.  
  – These changes are largely **out of end-user control** and create operational challenges for IT teams.

⚠️ **Updates can introduce new problems**  
  – Example: a payroll system update can change settings or **break compatibility** with company processes.  
  – Operational implication: **test new software or updates before deploying** them to users or production systems.

🕰️ **Running old software increases security risk**  
  – A **software bug** is an error that causes unexpected behavior; attackers exploit bugs in outdated software.  
  – Software updates typically include **critical security patches**, new features, and improved compatibility.  
  – Practical guideline: **update software regularly** to reduce exposure to vulnerabilities.

🧩 **Software management responsibilities in workplace setups**  
  – When provisioning a computer, install all **job-required software** so users can "hit the ground running."  
  – Remove **unnecessary or irrelevant software** to reduce clutter and potential risk.

🛡️ **Malicious software and source verification**  
  – Some installed software may be malicious and harm systems.  
  – Always verify that software **comes from a reputable source** before installing.

🔐 **Access controls on software installation**  
  – Common industry practice: **prevent users from installing software without administrator approval**.  
  – Mechanism: the system blocks installation and prompts for **administrator credentials** to proceed.  
  – Effect: reduces unauthorized or unwanted software installation.

🛠️ **Core software management tasks**  
  – **Installing** software.  
  – **Updating** software.  
  – **Removing** software.
