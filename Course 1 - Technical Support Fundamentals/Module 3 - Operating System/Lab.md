# Creating a Folder with Windows

## Creating a folder

### For keyboard-only users:

1. Navigate to the desktop using the Tab key and arrow keys.
2. Press Ctrl + Shift + N to create a new folder.
3. The new folder will be named New Folder by default.
4. Rename folder with My Super Cool Folder and press Enter.

### For other users:

1. Creating a folder in Windows is super simple. Right-click anywhere on the desktop and you should see this menu appear:

2. Hover your cursor over "New,"

3. Move your cursor to the first option ("Folder"), and select it. If your cursor leaves the menus,
   they may disappear; if this happens, just repeat the steps again. A new folder should appear on
   the desktop. It'll be called "New folder" by default, and the text should already be highlighted for you to edit.

4. f you click anywhere before changing the name, you'll need to right-click the folder and select the "Rename" option to be able to edit it. Change the name to "My Super Cool Folder".

5. Your folder has been created and named! To verify, double-click on the folder and a Windows Explorer window should pop up, showing you the contents of your currently empty folder.

# Creating a Folder with Linux

To create a sample folder, enter this command into the terminal:

```
mkdir my-super-cool-folder

```

This creates a directory called "my-super-cool-folder" in your current directory.
