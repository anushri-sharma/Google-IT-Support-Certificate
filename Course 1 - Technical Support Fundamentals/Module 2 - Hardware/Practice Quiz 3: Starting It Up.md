# Practice Quiz: Starting It Up

<img width="683" height="320" alt="image" src="https://github.com/user-attachments/assets/93b6db57-966c-4f64-86eb-4151436b209f" />

```
BIOS (Basic Input/Output System) starts when you power on the computer
It:
- Initializes hardware (CPU, RAM, storage)
- Runs POST
- Begins the boot process by loading basic drivers
```

```
POST (Power-On Self-Test) runs when the computer starts
It checks hardware like:
- RAM
- CPU
- Keyboard
Ensures everything is working before the system boots

```

<img width="738" height="360" alt="image" src="https://github.com/user-attachments/assets/b48febd7-fa62-4cb9-b02a-cbe66c532849" />

```
BIOS settings (like date, time, boot order) are stored in CMOS (Complementary Metal-Oxide-Semiconductor) memory
This memory is powered by a small battery on the motherboard, so settings are retained even when the computer is off
```
