🔥 **What RAM is and what it does (short-term memory, volatile)**  
  – **RAM** stores data the CPU needs for fast access; its contents change constantly and are not permanent.  
  – **Volatile**: when power is removed, RAM is cleared — unsaved work (documents, game progress) is lost.  
  – **Edge case**: while almost all consumer RAM is volatile, there are non-volatile variants (e.g., NVDIMM, battery-backed memory) used in special systems.

💡 **How programs use RAM**  
  – To run a program the system loads a copy of it into **RAM** so the **CPU** can process it.  
  – The amount of available RAM limits how many programs (and how large) can be run simultaneously.

📦 **What capacity labels mean (e.g., 16 GB RAM, 8 GB stick)**  
  – A spec like **16 GB** of RAM means the system can hold up to **16 gigabytes** of program data in RAM at once.  
  – RAM sticks are sold/labeled by **capacity** (e.g., **8 GB** per stick), not by the number of DRAM chips on them.

⚙️ **DRAM: how mainstream RAM stores bits**  
  – **DRAM (dynamic RAM)** represents each bit with a microscopic capacitor that is either charged (1) or discharged (0).  
  – These capacitors and transistors are fabricated as DRAM chips; chips are placed on memory modules that plug into the motherboard.

🧩 **Memory modules and physical form factors**  
  – **DIMM (Dual Inline Memory Module)** is the common form factor for desktop RAM; modules come in different physical sizes and **pin counts**.  
  – You must match the module type and pin/socket format to the motherboard; different RAM generations use different module pinouts.

⏱️ **SDRAM → DDR evolution and what it means**  
  – **SDRAM (Synchronous DRAM)** is synchronized to the system clock for faster coordinated transfers.  
  – **DDR (Double Data Rate SDRAM)** transfers data on both clock edges, increasing throughput; common iterations include **DDR1, DDR2, DDR3, DDR4**.  
  – **DDR** generations generally improve **speed**, **power efficiency**, and **maximum capacity** over previous versions.  
  – Clarification: **DDR5** is a newer generation beyond DDR4 and offers higher speeds and capacities than DDR4; RAM generation must match motherboard support.

🔌 **Compatibility and practical constraints**  
  – RAM must be **physically and electrically compatible** with the motherboard (correct module type, pin count, and supported DDR generation).  
  – Upgrading RAM requires checking motherboard specifications (supported DDR version, maximum capacity, and supported speeds).

⚡ **Performance implications**  
  – Faster RAM and newer DDR generations enable programs to run faster and support more concurrent tasks.  
  – Different DDR generations also affect **power consumption** and **maximum per-module capacity**, influencing battery life in laptops and upgrade paths in desktops.
