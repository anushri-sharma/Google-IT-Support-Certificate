
# [Virtualization](https://www.youtube.com/watch?v=1crD26vGBTE&t=6s)

**There are two ways you can run your services**

1. Dedicated hardware
2. Virtualized instance on a server.

# Virtualization in Infrastructure Services
## Running Services
- 🖥️ Dedicated hardware vs virtualized instances
- 🖧 Multiple virtual instances on one physical server

## Performance
- 🚀 Dedicated hardware offers better performance
- 🐢 Virtualized environment shares resources among many services

## Cost
- 💰 Dedicated servers are expensive for multiple services
- 💡 Virtualization allows many services on one physical server, reducing costs

## Resource Utilization
- 📊 Single service uses only 10-20% CPU on a server
- 🔄 Virtualization increases overall hardware utilization

## Maintenance
- 🛠️ Dedicated hardware requires downtime for updates
- ⚡ Virtualized services can be migrated or stopped quickly for maintenance

## Points of Failure
- ⚠️ Single physical machine failure impacts hosted services
- 🔄 Virtualization allows easy migration and backups across machines

## Redundancy
- 🛡️ Redundant physical servers prevent single points of failure
- 💾 Backup servers can be costly but increase reliability

## Making the Right Choice
- ⚖️ Weigh pros and cons of virtualization vs dedicated hardware
- 🎯 Choose the best option for your company needs
