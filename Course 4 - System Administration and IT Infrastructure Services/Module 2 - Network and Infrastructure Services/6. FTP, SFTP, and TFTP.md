
 # [FTP, SFTP, and TFTP](https://www.youtube.com/watch?v=hWAoxJd_Rjk&t=17s)

## Summary

We discusses network file transfer services, highlighting their roles, types, and use cases within organisational environments. It begins by emphasising the need for dedicated file transfer services over ad hoc methods like flash drives or simple remote copy tools. The content outlines several protocols, their characteristics, and practical applications, focusing on security and efficiency.
