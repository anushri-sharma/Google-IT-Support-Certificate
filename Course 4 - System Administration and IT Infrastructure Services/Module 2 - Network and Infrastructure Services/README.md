### We're going to learn about the common IT infrastructure services out there, and what you need to know to start integrating them into an organization.

### We'll also dig deeper into each infrastructure service individually.

### We will focus more on the physical infrastructure, services like servers, along with network infrastructure services that keep your company connected to the Internet.

### In short, we'll be servicing all the infrastructure services needs.


**Domain Name System (DNS)**: A global and highly distributed network service that resolves strings of
leers, such as a website name, into an IP address

**Intranet:** An internal network inside a company, accessible if you are on a company’s network

**NTP:** Network Time Protocol, keeping clocks synchronized on machines connected to a network

**Proxy server**: An intermediary between a company's network and the Internet, receiving network
trac and relaying that information to the company network

**Server operating systems**: Regular operating systems that are optimized for server functionality
