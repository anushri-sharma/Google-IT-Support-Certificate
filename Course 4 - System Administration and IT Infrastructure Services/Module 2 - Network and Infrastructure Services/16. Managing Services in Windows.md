
# [Managing Services in Windows](https://www.youtube.com/watch?v=fsdIGWUi7BM&t=2s)


Like Linux, Windows also allows the system administrator to manage the services that are running on the system.

For this example, let's look at the Windows update service.

This service is in charge of detecting software updates for either the operating system or other installed programs, downloading them and having them ready to be applied to the system.

**Let's get the status of the running service using the get service command.**

**Open Power Shell and go ahead and type in get service.**

<img width="1277" height="241" alt="image" src="https://github.com/user-attachments/assets/ee751b26-bad9-4564-a329-75148b2bacb6" />


Type in the shorthand form for Windows update service, which is this.

So, **wuauserv is a short name for the Windows update service.**

We can see that the windows update service is running and can get more information about it by running this next command.

Which is going to be **get service wuauserv and then I want to open format list.
Asterix.**

<img width="1279" height="558" alt="image" src="https://github.com/user-attachments/assets/2f4b23b0-c578-46ac-831b-e5af7b009dca" />

This will show us what type of service it is and how it's configured to run.

It's a good way to get additional information on the service you're interested in.

As with Linux, any user can query the status of the service, but only administrators can start or stop a service.

If you try to do the next steps with the normal user shell, you won't be able to run the commands.

Now, let's try stopping the service and then checking the status.

For this, I'll **open an administrative power shell** and **run the stop service command**.

<img width="1273" height="699" alt="image" src="https://github.com/user-attachments/assets/4e2dd4d1-ad7f-4c42-84c2-e2d06f9d2fc8" />

So I'm going to go into my start and instead of clicking on it, I'm going to right click and then type in run as administrator.

Yes, to the security control.

So now, I'm going to go ahead and **type in stop service wuauserve**.

<img width="1259" height="246" alt="image" src="https://github.com/user-attachments/assets/a9d19608-e065-47de-ac3d-fbbe6f62f140" />

Next I want to type in get service wuauserve.

So, this service has been stopped.

<img width="1280" height="336" alt="image" src="https://github.com/user-attachments/assets/8436f1d7-6797-4bc3-952a-060c42ea6d1c" />

In order to **start it back up**, we **execute the start service command**, which I'm going to do start service wuauserve to start the service.

And then I'm going to type in **get service, To show that the service is now running**.

<img width="1253" height="96" alt="image" src="https://github.com/user-attachments/assets/4881d0d7-8b55-479a-ac5b-83f34f2b8e34" />

Finally, you can list all services that are registered in the system using the get service command.

<img width="1269" height="680" alt="image" src="https://github.com/user-attachments/assets/a0c8cf18-f236-4b94-9040-26fe7a4b9a5f" />

We can also perform these same actions graphically using the services management console.

So I'm going to go ahead and click and start.

<img width="1269" height="680" alt="image" src="https://github.com/user-attachments/assets/6598bc0a-09b6-4d8c-b3e5-2629e0134558" />

This console shows us all the services in the system.

<img width="1273" height="674" alt="image" src="https://github.com/user-attachments/assets/c83a5311-4861-41e2-97b2-fb20458d47af" />

<img width="1279" height="715" alt="image" src="https://github.com/user-attachments/assets/237bd402-30b4-4131-aaab-e3564825e7a6" />

The ones that are running will say running in the status column, while the ones that aren't running won't say anything in the status column.

We can find the Windows update utility at the end of the list.

<img width="1275" height="719" alt="image" src="https://github.com/user-attachments/assets/8ffbfed6-d297-495a-829b-ebfb534123c4" />

We can stop it by right clicking on it.

And then clicking stop.

And then right clicking again, and hitting start.

And there you have it.

That's how you get the status, stop and start services in Windows.
