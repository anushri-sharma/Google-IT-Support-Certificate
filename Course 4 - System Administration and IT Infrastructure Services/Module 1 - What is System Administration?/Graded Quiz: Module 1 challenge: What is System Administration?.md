1. What device is used to control multiple computers with one keyboard, mouse, and monitor?
✅ KVM switch


KVM = Keyboard, Video (monitor), Mouse — exactly what it manages.



2. Which of the following tools is not provided by a KVM switch?
✅ Printer


A KVM switch only handles keyboard, monitor, and mouse—not printers.
