# Notes
 
**Summary of Web Servers and Data Storage for Online Services**:  
- Web services require web servers to respond to client requests by serving web pages.  
- However, web servers alone do not suffice for storing user information such as account details. It is critical to understand that storing customer data directly in file folders on web servers is highly insecure and improper.  
- Instead, **databases** are the standard solution for storing various types of data, including customer information, news articles, videos, large text blocks, images, and audio files.  
- Databases allow efficient storage, querying, filtering, and management of large volumes of data, which is essential for most web applications.

**Role of Database Systems in Web Applications**:  
- When building web products, data storage is typically handled by database systems rather than simple file storage on the server.  
- Database service refers to the database software running on servers that allows applications to **read from and write to the database**.  
- Commonly used database management systems include **MySQL** and **PostgreSQL**, both highly prevalent in web development, application building, and data analytics.  
- These systems require specialised knowledge of query languages and syntaxes (e.g., SQL) to effectively retrieve and manipulate data.

**Complexity and Specialisation in Database Administration**:  
- Managing databases is complex and requires specialised skills.  
- The role of a **database administrator (DBA)** is dedicated to overseeing database performance, integrity, security, backups, and recovery.  
- Mismanagement or data loss can have severe consequences for a company, which underlines the importance of expert administration.  
- Database administration is recognised as a specialised IT discipline focusing exclusively on maintaining the reliability and security of data storage systems.  

### Key Concepts and Definitions  

| Term                 | Definition                                                                                             |
|----------------------|----------------------------------------------------------------------------------------------------|
| Web Server           | Software and hardware that serves web pages to users upon request over the internet.                |
| Database             | Structured storage system designed to hold, query, and manage large volumes of data efficiently.   |
| MySQL                | Popular open-source relational database management system (RDBMS) used in many web applications.   |
| PostgreSQL           | Advanced open-source RDBMS known for extensibility and standards compliance.                         |
| Database Administrator (DBA) | IT specialist responsible for the installation, configuration, maintenance, and security of databases. |

### Core Insights  
- **Web servers alone do not store customer data securely or efficiently; databases are essential for this purpose.**  
- **Large-scale data management requires specialised database software and expertise in query languages.**  
- **Database administration is a critical IT function dedicated to preventing data loss and ensuring consistent database performance.**  

### Summary Table of Roles and Technologies  

| Component           | Purpose                                      | Examples                      | Required Skills                          |
|---------------------|----------------------------------------------|-------------------------------|-----------------------------------------|
| Web Server          | Serve web pages to clients                    | Apache, Nginx                  | Basic server administration             |
| Database System     | Store, query, and manage large data sets     | MySQL, PostgreSQL              | SQL, database design, query optimisation |
| Database Administrator | Maintain database integrity and security   | Not software-specific          | Backup strategies, security, performance tuning |
