**Introduction to Web Services and Troubleshooting via HTTP Status Codes**  
- Web services, like any machines, are prone to breakage and can involve multiple variables when troubleshooting.  
- This session does not cover specific troubleshooting scenarios but introduces **HTTP status codes** as simple diagnostic tools.  
- When a browser requests a site (e.g., google.com), the web server responds with an **HTTP response**.  
- This response often includes a **status message or code** indicating success, errors, or other information about the request.


**Definition and Importance of HTTP Status Codes**  
- **HTTP status codes** are numeric codes indicating errors or informational messages when accessing web resources.  
- They are essential for troubleshooting website issues as they provide clues to isolate the root cause.  
- A widely recognised example is the **404 Not Found** error, signalling that the requested URL does not correspond to any resource on the server.

**Example of a 404 Error**  
- Typing a non-existent URL such as `google.com/asdf` results in a **404 error message**: "The requested URL /asdf was not found on this server."  
- This confirms the URL does not point to any hosted content.  
- To verify the exact HTTP status code returned, it is necessary to inspect the raw HTTP response rather than relying solely on webpage error messages.

 
**Using Browser Developer Tools to Check HTTP Status Codes**  
- Modern browsers include developer tools that assist in diagnosing web and browser issues.  
- In Chrome, this tool is accessed via **Tools > Developer Tools**.  
- Developer Tools open alongside the browser window and provide multiple tabs for debugging and testing.

**Step-by-Step: Viewing HTTP Status Codes in Chrome Developer Tools**  
- Navigate to the **Network tab** within Developer Tools.  
- Refresh the page to capture network requests.  
- Locate the specific request (e.g., `google.com/asdf`) on the left panel.  
- Clicking the request shows detailed information, including the **HTTP status code**, which in this case is **404 Not Found**.  
- **4xx status codes** indicate client-side errors, such as invalid URLs or unauthorized access attempts.

**Other Common HTTP Status Code Categories**  
- **5xx status codes** signify server-side errors, meaning the web server hosting the content is experiencing problems.  
- Server administrators typically investigate and resolve these issues.  
- **2xx status codes** indicate successful requests where the server has processed the request correctly.  
- Understanding these categories helps identify whether issues arise from the client or server side.

**Summary and Practical Advice**  
- HTTP status codes provide valuable insights beyond just error notifications.  
- If encountering unfamiliar status codes, searching for their definitions online will often clarify the issue.  
- This knowledge empowers users and developers to better diagnose web service problems effectively.

### Summary Table: HTTP Status Code Categories and Meanings

| Status Code Range | Meaning                       | Example         | Description                                         |
|-------------------|-------------------------------|-----------------|-----------------------------------------------------|
| 2xx               | Success                       | 200 OK          | Request was successful, content delivered correctly |
| 4xx               | Client Error                  | 404 Not Found   | Client made a bad request, e.g., invalid URL         |
| 5xx               | Server Error                  | 500 Internal Server Error | Server failed to fulfill a valid request           |

### Key Insights  
- **HTTP status codes are essential diagnostic tools in web troubleshooting.**  
- **4xx codes point to client-side issues, 5xx codes to server-side problems, and 2xx codes confirm success.**  
- Browser developer tools offer a direct method to inspect HTTP responses and status codes.  
- Familiarity with common HTTP codes significantly aids in isolating and resolving web errors.
