

# [What are file services?](https://www.youtube.com/watch?v=NqgVkFyCQsE&t=2s)

**File storage services** allow us to centrally store files and manage access between files in groups.

You can set up a file storage server that will let users access a shared directory to modify, or add files and much more.

The other way to maintain a file storage service, is by using a **Cloud file storage provider**.

There are lots of providers that offer secure and easily managed file storage.
