Adware is designed to automatically deliver advertisements to your screen, often redirecting your browser or popping up windows to generate revenue for the author.

4. A rootkit

A rootkit is a collection of software tools that allows an attacker to gain privileged (admin-level) access to a computer while actively hiding its presence from the operating system and antivirus software.

10. Social Engineering
Baiting
Baiting is a social engineering attack where an attacker leaves a physical
"lure" (like a USB drive or SD card) in a public place, relying on human curiosity
to get the victim to plug it into a secure system.

3. Adware
If your browser is jumping to product pages or showing unwanted pop-ups, it's Adware.
It’s usually more annoying than destructive, but it can slow your system down.

4. A backdoor
A backdoor is a method of bypassing normal authentication in a cryptosystem or algorithm,
often left by a developer or installed by an attacker to ensure they can get back into a
system later.

5. A rogue AP (Access Point) attack
This is a classic example of an "insider threat" (even if accidental) creating
an opening for an external attacker.

6. A SYN flood attack
A SYN flood is a type of Denial of Service (DoS) attack that exploits the TCP
handshake process to overwhelm a server.

7. Overwhelming a System (Select all that apply)
Ping flood: Sending a massive amount of ICMP Echo Request packets.

SYN flood: Exploiting the TCP three-way handshake to leave connections
"half-open" and exhaust server resources.


8. Injection and Web Attacks
Cross-site scripting (XSS) attacks
XSS allows an attacker to execute scripts in the victim's browser. Since cookies
are often stored in the browser, these scripts can steal "session cookies" to let
an attacker impersonate the user.

9. Password Security
A brute force attack
A brute force attack is the "guess everything" method. It differs from a
dictionary attack, which only tries common words or previously leaked passwords.

10. The Human Element
Social engineering
Social engineering attacks don't target software vulnerabilities; they target human
vulnerabilities like trust, curiosity, or the desire to be helpful.
