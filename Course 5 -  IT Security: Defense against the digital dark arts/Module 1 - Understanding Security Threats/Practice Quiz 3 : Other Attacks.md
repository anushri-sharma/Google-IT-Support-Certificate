Here are the correct answers and explanations for your security quiz:

1. Access Control
Badge readers
Badge readers are the standard physical security device used to grant employees access to buildings and restricted zones using RFID or NFC cards, often logging the time of entry.

Bollards are short posts used to prevent vehicles from driving into buildings.

Equipment locks secure physical hardware like laptops to desks.

2. Protecting Against Injection

Use data sanitization.

Use input validation.
Data sanitization cleanses data by removing or neutralizing dangerous characters,
while input validation ensures the data matches expected formats (like a date or a number)
before it is processed. Strong passwords protect accounts, but they don't stop a
code injection vulnerability.

3. Physical Security Breach

A tailgating attack

Tailgating occurs when an unauthorized person follows an authorized person into a secure
area without their own credentials. This often relies on social engineering or simply
"blending in" (like dressing as a repair worker).

4. Injection Attacks

An attack that inserts malicious code into a website and targets the users of a service.
XSS is unique because the "victim" is not the web server, but the user visiting the site.
The attacker injects a script (usually JavaScript) that runs in the visitor's browser.

Note: The option involving database commands describes a SQL Injection.

5. Password Attacks

A brute-force attackA brute-force attack is the "shotgun" approach where
software tries every possible combination ($aaa1$, $aaa2$, etc.) until it finds the right one.
A dictionary attack is more targeted, only trying common words and phrases from a pre-compiled list.
