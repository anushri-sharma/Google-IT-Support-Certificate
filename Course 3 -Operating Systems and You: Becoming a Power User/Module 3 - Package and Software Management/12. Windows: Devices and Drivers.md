🔧 Microsoft **Device Manager** organizes all devices and drivers on a Windows PC into a single management console.  
  – Open it via **devmgmt.msc** in the Run dialog or by right‑clicking **This PC → Manage → Device Manager**.  
  – Devices are grouped by broad categories (example: displays appear under **Monitors**).  
  – Grouping and initial device entries usually occur automatically via the **Plug and Play** system.

🔌 How Windows detects and installs a new device (Plug and Play flow).  
  – When a device is connected, Windows asks the device for its **hardware ID** (a vendor‑assigned identifier).  
  – Windows uses that **hardware ID** to search for the correct driver in this order:  
  –  – **Local list of well‑known drivers** first.  
  –  – **Windows Update** or the **driver's store** if the local list doesn't suffice.  
  –  – A supplied **installation disk** or manually provided driver location if available.  
  – After locating a suitable driver, Windows installs it so the device can be used (examples: keyboard, mouse, monitor).

⚙️ Direct management of devices and drivers inside **Device Manager**.  
  – Expand any category to view individual devices.  
  – Right‑click a device to access actions: **Uninstall**, **Disable**, **Update** driver, and **Scan for hardware changes**.  
  – Choosing **Properties** shows device details such as **manufacturer** and **driver version**.
