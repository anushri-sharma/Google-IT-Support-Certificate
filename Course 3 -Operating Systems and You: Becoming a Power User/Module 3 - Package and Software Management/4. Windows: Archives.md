🔥 **What an archive is**
  - **Archive = one or more files compressed into a single file** for transport or storage.
  - Archives can contain **source code (package archives)** or any other data (pictures, music, documents).
  - Installing software from a source archive is called **installing from source**; the archive is just the compressed source files.

📦 **Common archive formats**
  - **.tar, .zip, .rar** are frequently encountered formats.
  - Different formats may require different extraction tools or steps.

🔧 **Extract vs install from source**
  - First step with any archive: **extract the archive to view the files inside**.
  - After extraction, **installation method depends on the programming language or build system** used by the source (varies widely); extraction is the universal prelim step.

🧰 **7‑Zip: a widely used tool on Windows**
  - **7‑Zip** is an open‑source archiver that handles many formats (including **.zip, .rar, .tar**).
  - Typical GUI actions:
    - To extract: right‑click archive → **7‑Zip → Extract Here** (or other extract options).
    - To create an archive: place files in a folder → right‑click folder → **7‑Zip → Add to archive** → OK.

✉️ **Why archive files**
  - Combine multiple files into **one file** for simpler sending (email, transfer) and organized storage.
  - Compression often reduces file size and transmission time.

> 💡 **Practical demo example (from transcript)**
  - An archive named **colors.zip** was extracted (revealing multiple files).
  - A folder **new colors** was created, files were added, then archived using 7‑Zip via **Add to archive → OK**.

⌨️ **PowerShell (command‑line) archiving — requires PowerShell v5 or newer**
  - PowerShell can **compress and extract archives from the CLI**.
  - Example compress command (to create a Zip from a folder):
    - **Compress-Archive -Path "Cool files" -DestinationPath "CoolArchive.zip"**
    - This takes everything in the **"Cool files"** directory and creates **CoolArchive.zip** on the desktop (or specified destination).
  - Corresponding extract command:
    - **Expand-Archive -Path "CoolArchive.zip" -DestinationPath "TargetFolder"**
    - Use when you need to extract via script or CLI.

⚠️ **Edge cases and practical notes for IT Support**
  - Many archive types and variants exist; not every format is natively supported by Windows Explorer.
  - Use a universal tool (like **7‑Zip**) or PowerShell for common automation tasks.
  - Always confirm the archive format before choosing an extraction tool; installation from source requires additional language/build steps after extraction.
