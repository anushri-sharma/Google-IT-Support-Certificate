
# [Linux: Adding and Removing Users](https://www.youtube.com/watch?v=K6Bly7I0TLY&t=2s)

#### To add a new user in Linux, you can use the user add command, sudo useradd juan.

<img width="1273" height="358" alt="image" src="https://github.com/user-attachments/assets/9902c5cb-45d4-4f3f-b959-362ff1429879" />

You can combine this with the password command to make the user change their password on login.

#### To remove a user, you can just use sudo userdelete juan.
