```

In Linux permissions:

First trio → file owner
Second trio → group
Third trio → others (everyone else)

So in:
dr-x-wxr--

r-x → owner
-wx → group
r-- → others
```

In the ls -l output, the first character indicates the file type:

d → directory ✔️
- → regular file
