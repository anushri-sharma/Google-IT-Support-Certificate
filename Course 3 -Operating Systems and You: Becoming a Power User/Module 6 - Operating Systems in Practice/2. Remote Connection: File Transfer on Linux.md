There are lots of ways to transfer files.

**SCP or secure copy is a command you can use in Linux to copy files between computers on a network.**

**It utilizes SSH to transfer the data.**

Just like you would SSH into a machine, you can send a file that way.

we want to copy over a file from our computer to another computer.

To do this, we can run the SCP command with a few flags.

**In this command, we run the SCP command with the path of the file we want to
transfer to the user account, host name, and path where we want to copy the file to.**
