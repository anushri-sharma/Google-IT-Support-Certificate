
# [Windows: Mounting and Unmounting a Filesystem](https://www.youtube.com/watch?v=RgTCT7JnovI&t=1s)

Mount your file system to a drive.

In this case, we want to make our USB drive accessible, so we mount the file system to drive.

Windows does this for us automatically.

<img width="1267" height="338" alt="image" src="https://github.com/user-attachments/assets/0c55d820-e205-41de-a30a-cc89a36eda2b" />

If you plug in a USB drive, it'll show up under your list of drives and you can start using it right away.

When you're done using the drivers, we'll just have to safely eject or essentially unmount the drive by right-clicking and selecting Eject.
