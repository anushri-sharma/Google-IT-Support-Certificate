Without a file system, the operating system wouldn't know how to organize files.

For **Windows**, we use the **NTFS file system** and for **Linux** it's recommended to use **ext4**.

<img width="1272" height="309" alt="image" src="https://github.com/user-attachments/assets/91e1522e-ef5e-41ff-a431-b0a585a1ee73" />


**File systems have different compatibility with different OSS**, most of the time, 
cross operating system support is minimal at best.

Let's say you have a **USB drive that's using an NTFS file system.**
**Both Windows and Linux's Ubuntu** can **read** and **write to the USB drive**

<img width="1279" height="529" alt="image" src="https://github.com/user-attachments/assets/97c5b3af-b0b4-4c29-b8a6-e4e096c61c6c" />


If you have an **ext4 USB drive**, it'll **only work on Ubuntu**, and not on windows.

<img width="1273" height="586" alt="image" src="https://github.com/user-attachments/assets/645da961-53bb-4319-a9e7-ef59cd176a6a" />


**Foul systems like FAT32** that support reading and writing data to **all three major operating systems**.
