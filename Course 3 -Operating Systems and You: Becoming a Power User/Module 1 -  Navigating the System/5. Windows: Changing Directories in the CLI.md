
# [Windows: Changing Directories in the CLI](https://www.youtube.com/watch?v=FozvVCIMn0g&t=5s)

🔥 **Prompt shows current directory; PowerShell typically opens in your home directory**
  
  - **Home directory** is where the shell usually starts (e.g., C:\Users\Cindy).
  - The prompt itself reflects the **current working directory**.

<img width="1277" height="421" alt="image" src="https://github.com/user-attachments/assets/89502e26-34a1-4ea5-a04b-d16fd7bea386" />

💡 **Check current directory with `pwd` (print working directory)**
  
  - Run **pwd** to display the full path of where you are.

📁 **Change directory with `cd` (change directory)**
  
  - **cd ** switches the shell to the specified directory.
  - **Path types:** **absolute** (full path starting with drive letter, e.g., C:\Users\Cindy\Documents) or **relative** (path described from your current location).

📌 **Absolute path example**
  
  - **cd C:\Users\Cindy\Documents** moves directly to that folder by full path.

<img width="1280" height="183" alt="image" src="https://github.com/user-attachments/assets/600de373-8695-4239-b6ac-10d24744b537" />

↩️ **Go up one level with `cd..` (parent directory)**
  
  - **cd..** moves to the parent of the current directory (relative path).
  - Example sequence: from C:\Users\Cindy\Documents run **cd..** → now in **C:\Users\Cindy**.

<img width="1247" height="444" alt="image" src="https://github.com/user-attachments/assets/eba6df64-001e-414d-8b08-dc194fad49d9" />


🧭 **Combine relative moves and subfolders (shorter paths)**
  
  - From the home folder you can do **cd..** then **cd Desktop**, or do both in one step: **cd..\** (e.g., **cd..\\Desktop**).
  - The prefix **.\** (dot backslash) denotes the **current directory**, so **.\Desktop** explicitly means the Desktop inside the current folder.

<img width="1278" height="425" alt="image" src="https://github.com/user-attachments/assets/8b5bd8c7-5ef2-4eae-901d-3ce1a30f1514" />


🏠 **Tilde (`~`) is a shortcut for the home directory**
  
  - **cd ~\Desktop** moves to the **Desktop** inside your home folder without typing the full home path.

<img width="1280" height="128" alt="image" src="https://github.com/user-attachments/assets/98de4df8-2702-4e9c-9d96-e7ed6f845724" />


⌨️ **Tab completion auto-completes names and cycles options**
  
  - Type the first letters and press **Tab** to autocomplete a file or folder name (reduces typing and typos).
  - Press **Tab** repeatedly to rotate through all matches (e.g., typing `D` then Tab cycles **Desktop**, **Documents**, **Downloads**).
  - Narrowing the prefix (e.g., `De`) reduces matches so Tab can directly complete **Desktop**.
