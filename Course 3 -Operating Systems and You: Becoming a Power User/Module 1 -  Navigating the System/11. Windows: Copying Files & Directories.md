
#  [Windows: Copying Files & Directories](https://www.youtube.com/watch?v=TJ6raKkCc-8&t=3s)

🔧 GUI copy methods  
–  **Right-click → Copy → Paste** in Windows File Explorer.  
–  **Hotkeys:** **Ctrl-C** to copy and **Ctrl-V** to paste.

🖥️ PowerShell copy command  
–  Use **cp** (PowerShell alias of **Copy-Item**) to copy files and directories.  
–  Specify a **source** and a **destination**, e.g., `cp mycoolfile.txt Desktop` copies **mycoolfile.txt** to **Desktop**.  
–  Enclose paths with spaces in quotes, e.g., `cp "Bird Pictures" Desktop`.

✳️ Wildcard selection for multiple files  
–  The **asterisk (*)** wildcard matches any sequence of characters; use it to select files by pattern.  
–  Example: `cp *.jpg Documents` copies all files with the **.jpg** extension (e.g., hotdog.jpg, cotton_candy.jpg, pretzel.jpg) to **Documents**.  
–  Wildcards let you copy many files at once instead of repeating commands.

📁 Copying directories and recursion  
–  Running `**cp "Bird Pictures" Desktop**` without recursion copies the directory entry but **not** the directory’s contents.  
–  Add **-Recurse** to include all files and nested subdirectories.  
–  Example: `cp "Bird Pictures" Desktop -Recurse` copies the folder and every file/subdirectory inside it.  
–  **-Recurse** works by listing directory contents and repeating that listing for each subdirectory (i.e., it recurses into subfolders).

it doesn't do is copy over the contents of the directory.

**To copy over the contents of the directory,** you need to use another command parameter, -Recurse.

The **dash recursive parameter** lists the contents of the directory.

Then if there are any subdirectories in that listing, it'll recurse or repeat the directory listing process for each of those subdirectories.

We'll need to use the **-Recurse parameter** with copy to copy the contents of the directory along with the directory itself.

We're going to use a **new parameter, -Verbose.**

Copy doesn't output anything to the CLI by default unless there are errors.

When we use **copy-Verbose**, it'll output one line for each file the directory being copied.

📝 Verbose output and default silence  
–  **Copy-Item/cp is silent by default** and only outputs on errors.  
–  Use **-Verbose** to emit a line for each file copied so you can see progress.  
–  Example: `cp "Bird Pictures" Desktop -Recurse -Verbose` shows each copied file.

⚠️ Practical tips and cautions  
–  Use quotes for any path containing spaces (e.g., `"Bird Pictures"`).  
–  Be deliberate with wildcards to avoid unintentionally selecting files.  
–  Remember that copying a directory without **-Recurse** will not transfer its contents; include **-Recurse** when you need nested files.
