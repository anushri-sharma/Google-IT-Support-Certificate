
# [Linux: List Directories](https://www.youtube.com/watch?v=dFZp1tR-L9U&t=5s)

🔥 **Root directory and path format**  
  – **Root directory is /** (a single forward slash).  
  – Example absolute path: **/home/cindy/Desktop** (Linux uses forward slashes).  
  – Windows equivalent example: **C:\Users\cindy\Desktop** (uses backslashes).  
  – Running **ls /** lists the contents under the root directory.  
  – Running **ls** with no path lists the **current directory**.

🧭 **ls command — purpose and usage**  
  – **ls** lists directory contents; give a path to list a specific directory or none to list the current directory.  
  – The **ls** binary is commonly located in **/bin** (an essential binaries directory).

📁 **Important directories under /**  
  – **/bin** — stores essential binaries/programs (example: **ls**).  
  – **/etc** — stores system configuration files.  
  – **/home** — users’ personal directories (documents, pictures, etc.).  
  – **/proc** — virtual filesystem with information about currently running processes.  
  – **/usr** — contains user-installed software and shared read-only data (does not hold user home files).  
  – **/var** — holds variable data such as system logs and files that change frequently.

⚙️ **Command flags (options) and help resources**  
  – Flags modify command behavior and are usually prefixed with a hyphen (e.g., **-l**).  
  – Some options use double hyphens (example: **--help**).  
  – Use **--help** to display a quick list of options and descriptions (reference output, not memorization).  
  – Use **man** (**manual pages**) for **more detailed documentation** (e.g., **man ls**).

🧾 **Key ls flags demonstrated and details**  
  – **-l** (long): shows detailed file/folder information in long-list format.  
    – – **Columns explained (left → right):** file permissions; number of links; file owner; file group; file size; last modification timestamp; filename.  
    – – **Number of links** refers to the count of hard links to the file (directory link counts reflect entries).  
    – – **Group** indicates the group ownership used for access control.  
  – **-a** (all): includes hidden files in the listing.  
    – – Hidden files are files or directories whose names begin with a dot (e.g., **.iamhidden**).  
    – – **-a** also makes **.** (current dir) and **..** (parent dir) visible.  
  – Flags can be combined (short options): **ls -l -a**, **ls -la**, and **ls -al** are equivalent for these flags; ordering of these flags does not change the resulting output for these options.

🔎 **Practical implications and tips**  
  – Use **ls --help** or **man ls** when you need to discover additional options (sorting, recursion, human-readable sizes, colorized output, etc.).  
  – For readability of file sizes, pair **-l** with options such as **-h** (human-readable) when available (see man page).  
  – Inspect **/proc** via ls to quickly peek at running-process info (process directories are numeric).
