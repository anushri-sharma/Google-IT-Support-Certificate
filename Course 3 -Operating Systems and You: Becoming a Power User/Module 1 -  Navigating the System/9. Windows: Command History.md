
# [Windows: Command History](https://www.youtube.com/watch?v=b16wBtBwuMg&t=1s)

🔥 **Command history concept**  
  - **PowerShell** records every entered command **in memory** and appends it to a **persistent history file**, allowing reuse later.

🧭 **View recent commands**  
  - Use the **history** command to list previously entered commands.  
  - The raw list is informational; practical reuse requires navigation or searching.

⬆️⬇️ **Navigate and reuse previous commands**  
  - Press the **Up/Down arrow keys** to cycle through past commands.  
  - You can recall a command, edit it (for example append a suffix) and run it again instead of retyping.  
  - Example: recall **mkdir my cool folder**, append **2**, then run to create a new folder.

<img width="1280" height="661" alt="image" src="https://github.com/user-attachments/assets/20782e3e-7c7c-4d7e-b170-2d49c3b7baa8" />


🔎 **Search past commands interactively**  
  - Press **Ctrl-R** and type parts of a command to get incremental matches from history.  
  - Type a fragment (e.g., **folder**) to surface related previous commands like **mkdir** entries.

<img width="1276" height="108" alt="image" src="https://github.com/user-attachments/assets/4204f1a3-61e5-4c97-b3f5-fdb08b56a89f" />


⚠️ **Fallback for older PowerShell versions**  
  - If **Ctrl-R** is unavailable, type **#** followed by a fragment of the old command, then use **Tab** to cycle matching history entries.  
  - This leverages **tab completion** to iterate history matches.

<img width="1280" height="73" alt="image" src="https://github.com/user-attachments/assets/dbfd513d-5d2e-4f2d-a2b5-e21516b2b2a2" />

🧰 **Recommended complementary tools**  
  - Use **tab completion** to speed typing and cycle candidates.  
  - Use **Get-Help** (or **get-help**) to look up cmdlet usage and options.  
  - Combined, **history + tab completion + Get-Help** significantly speed command-line workflows.

🧹 **Clearing the screen**  
  - Use **clear** to remove output from the current shell view; this does **not** delete your command history.
