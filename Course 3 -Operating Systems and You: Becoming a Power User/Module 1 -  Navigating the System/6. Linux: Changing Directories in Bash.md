
# [Linux: Changing Directories in Bash](https://www.youtube.com/watch?v=S0Bb90LFaSU&t=2s)

🔎 Current location check  
  - **pwd** (print working directory) shows the current path; example output in the video: **/home/cindy/desktop**.  
  - The shell also exposes **$PWD** as the environment variable holding the same path.

<img width="1278" height="162" alt="image" src="https://github.com/user-attachments/assets/fe6b4b62-729b-4e1e-ad63-31f94c654882" />


📁 Change directories with cd  
  - Use **cd** to move between folders.  
  - **Absolute path** example: **cd /home/cindy/documents** — goes directly to that full path.

<img width="1280" height="171" alt="image" src="https://github.com/user-attachments/assets/52dbd659-ace3-4d13-8dfb-ab2e429f2b43" />

  - **Relative path** example: **cd ../documents** — **..** moves up one directory, then into **documents**.

<img width="1280" height="172" alt="image" src="https://github.com/user-attachments/assets/b6259ceb-f08f-4db6-bfd2-9e711a417e32" />

🏠 Home directory shorthand (~)  
  - The **tilde (~)** expands to your home directory.  
  - Example: **cd ~/desktop** takes you to **/home/yourusername/desktop** (in the video: **/home/cindy/desktop**).

<img width="1280" height="173" alt="image" src="https://github.com/user-attachments/assets/48ab3a15-9692-44d9-a320-973910f0c4b3" />


🧭 Tab completion behavior in Bash  
  - Bash supports **tab completion** to auto-complete filenames and directories.  
  - If multiple matches exist, Bash **lists all options at once** rather than cycling through them.

<img width="1276" height="277" alt="image" src="https://github.com/user-attachments/assets/b27558fc-a436-4a94-92a6-1841aae2ccfd" />


🔗 Similarities and portability between PowerShell and Bash  
  - Core navigation commands (**pwd**, **cd**) are the **same** conceptually in both environments, easing transfer of skills.  
  - A notable behavioral difference: **tab completion** interaction differs (Bash lists options; Windows PowerShell may cycle through matches).

⚠️ Practical nuance (Linux specifics to be aware of)  
  - **Paths are case-sensitive** on most Linux systems (e.g., **Desktop** ≠ **desktop**), so use exact casing.
