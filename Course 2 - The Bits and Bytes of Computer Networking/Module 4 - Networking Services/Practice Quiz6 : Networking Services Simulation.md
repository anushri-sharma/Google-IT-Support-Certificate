💡 Explanation:

NAT (Network Address Translation) allows:

- Use of private (non-routable) IP addresses inside a network
- Translation to a public IP address when accessing the Internet

This helps conserve public IP addresses and solves IP exhaustion issues.

❌ Why others are incorrect:

- VPN ✖️ → Secure connection, not IP management
- DNS ✖️ → Resolves names to IPs
- DHCP ✖️ → Assigns IP addresses, but doesn’t solve public IP shortage
